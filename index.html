Featured Projects

Sales Forecasting System

Problem: SKU-level monthly demand variance caused stockouts and overstocks.

Approach: Hybrid model (LightGBM + Prophet residuals), calendar & promo features, per-SKU holdout evaluation.

Result: MAPE 6.2% on validation, reduced stockouts by estimated 12%.

Stack: Python, pandas, LightGBM, Prophet, Docker, Postgres

Link: repository / demo

Retention Dashboard & Cohort Analysis

Problem: Rising churn with limited channel-level insight.

Approach: Cohort analysis, RFM segmentation, dashboard with cohort selector & channel breakdown.

Result: Identified 3 at-risk cohorts and recommended onboarding flows; projected revenue uplift +9%

Stack: SQL, Tableau, Python

Link: dashboard demo

Customer Segmentation for Targeted Campaigns

Problem: Generic promotions had low ROI.

Approach: Feature engineering (recency, frequency, monetary, behavior), KMeans + HDBSCAN exploration, LTV estimation.

Result: Targeted campaigns increased conversion in segment A by 18%.

Stack: Python, scikit-learn, SQL

Link: analysis notebook

Realtime Monitoring & Alerting

Problem: Delayed detection of ETL failures and KPI drift.

Approach: Lightweight ETL validation, automated alerts, dashboarding of data quality metrics.

Result: Mean time to detect decreased from 4h to 15m.

Stack: Airflow, Python, Prometheus, Grafana
