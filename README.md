분석 과정

데이터 수집 (SQL)

6개월간 고객 거래 데이터 50만 건 추출
회원가입, 구매이력, 접속로그, 고객센터 문의 테이블 JOIN


데이터 전처리 (Python)

결측치 처리: 구매금액 중앙값 대체, 이상치 IQR 방식 제거
파생변수 생성: RFM 점수, 구매빈도, 평균구매금액, 최종구매일


탐색적 데이터 분석 (Python + Excel)

코호트 분석: 가입 월별 리텐션율 추적
Excel 피벗테이블로 제품 카테고리별 재구매율 계산
상관관계 분석: 할인쿠폰 사용과 재방문율 관계 파악


시각화 대시보드 (Tableau)

실시간 고객 세그먼트 모니터링 (충성, 잠재이탈, 이탈고객)
월별 매출 추이 및 이탈률 트렌드 시각화
제품 카테고리별 전환율 퍼널 차트
